Projects with known vulnerabilities that have been fixed

The following is a list of open source projects which had known security vulnerabilities which have now been fixed. In many cases, copies of those projects were made before the vulnerability was fixed. In some cases, the copy was fixed, but in other cases the vulnerability is still present in the copy.

Each entry in the project list below contains 4 lines:

  1. The name of the project and link to the project's page. This is the project where the vulnerability was fixed.
  2. A link to the CVE (Common Vulnerabilities and Exposures) list entry. This gives information about the vulnerability and the fix. The CVE page also includes a link (near the top of the page) to the National Vulnerability Database (NVD) which gives more information about the vulnerability.
  3. A link to a description of the fix.
  4. A link to a list of active projects which contain the vulnerability.
  5. A link to the full report. (not needed for this assignment).
